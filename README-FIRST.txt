QUANTA15 — GITHUB + VERCEL

1. Extract the ZIP.
2. Create a new empty GitHub repository named quanta15.
3. Upload everything INSIDE the Quanta15-Vercel-GitHub folder.
4. Commit the files to main.
5. Sign in to Vercel with GitHub.
6. Add New → Project → import quanta15.
7. Leave the detected Next.js settings unchanged and select Deploy.
8. Open the Vercel address and select Feed Health.

The Vercel backend route at /api/live fetches public market data server-side, avoiding browser CORS blocks.
No API keys are needed for this read-only version.
