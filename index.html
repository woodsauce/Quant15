One-time setup helper
  1. Create a Coinbase Advanced Trade / CDP API key with View and Trade permission only.
  2. Copy the key name that looks like organizations/.../apiKeys/....
  3. Paste the EC private key below only to convert it to Base64 locally in this browser. It is not saved by the app.
  4. Add the generated values to Vercel Environment Variables, redeploy, then press Test Coinbase Key.
